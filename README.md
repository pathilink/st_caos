# st_caos

Análise exploratória de dados sobre o engajamento de funcionários em uma empresa.

## Instalação 

Use o gerenciador de pacotes [pip](https://pip.pypa.io/en/stable/) para instalar o modulo nltk.

```bash
sudo apt install python3-nltk
```

## Uso

```bash
jupyter notebook
```
